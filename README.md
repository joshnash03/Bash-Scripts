Desolate: Deletes all files and subdirectories in a given directory
  Usage: "desolate /path/to/target/directory/" (Must be a valid directory)

Populate: Creates a given number of .txt files of the same name
  Usage: "populate /path/to/target/directory/" (Must be a valid directory); enter number of files; enter base file name
